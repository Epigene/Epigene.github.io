In Rails controllers you have access to an instance of `ActionDispatch::Request` object.

```ruby
request #=> loads of data.
```

You can access both this object's instance variables and some methods.

Here I will provide a link to the full list of variables and methods as well as a discussion of the most useful of them.

I debugged the Request object's instance methods with

```ruby
request.class.instance_methods(false).sort
```
